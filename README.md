BrightTV Customer Value Management Analysis
* Project Overview

This project focuses on analyzing user behavior and viewing patterns for BrightTV, with the goal of supporting the Customer Value Management (CVM) team in growing the company’s subscription base for the current financial year.

The analysis is based on user profile data and viewer transaction data, where each record represents a unique viewing session.

* Objective

To provide actionable insights that will help:

Increase user engagement and content consumption
Identify key drivers of user behavior
Recommend strategies to grow BrightTV’s subscriber base
* Dataset Description

The dataset includes:

User profile information
Viewer transaction/session data
Timestamps recorded in UTC (converted to South African time - SAST)

Important Notes:

Each record represents one viewing session
Time-based analysis required conversion from UTC → SAST
* Key Analysis Areas
1. User & Usage Trends
Daily and monthly viewing patterns
Peak vs off-peak consumption periods
Active user distribution
Session frequency and duration trends
2. Factors Influencing Consumption

Analysis explored how the following impact user engagement:

Time of day (morning, afternoon, evening)
Day of the week (weekday vs weekend)
Content type and category
User activity patterns
3. Content Recommendations

To improve engagement on low-consumption days:

Promote high-performing content during off-peak periods
Introduce targeted content releases (e.g., weekends or midweek dips)
Personalize recommendations based on user behavior
Highlight trending or popular content
4. Growth Strategies for User Base

Recommended initiatives include:

* Personalized marketing campaigns
* Referral and loyalty programs
  * Exclusive and original content offerings
* Improved user experience and platform accessibility
  * Data-driven content recommendations
